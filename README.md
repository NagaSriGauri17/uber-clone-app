# 🚖 Uber Clone App

A full-stack Uber-like mobile application built using React Native and Expo, replicating real-world ride booking functionality.

---

## 📱 Features (Planned & In Progress)

- Ride booking with pickup & destination input  
- Real-time route visualization using Google Maps  
- Distance-based fare estimation  
- Ride history tracking  
- User authentication (Clerk / JWT)  
- Payment integration (Stripe)  

---

## ⚙️ Tech Stack

### Frontend
- React Native (Expo)
- TypeScript
- Tailwind CSS (NativeWind)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### APIs & Integrations
- Google Maps API
- Google Places API
- Stripe (Payments)

### State Management
- Zustand

---

## 🧠 Architecture

React Native App → Node.js Backend → MongoDB

---

## 🚧 Project Status

Currently under development. Core UI and backend integration in progress.

---

## 🎯 Goal

To build a production-level ride-booking application similar to Uber with scalable architecture and real-time features.

---

## 📌 Note

This project is inspired by modern ride-sharing applications and extended with a custom backend using MongoDB instead of PostgreSQL.
